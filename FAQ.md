

### **Production build**

<pre>
    <code>
        npm run build
    </code>
</pre>
<br />

Builds the app for production to the `build/prod` folder.<br />

### **Future Development build**

<pre>
    <code>
        npm run dev
    </code>
</pre>
<br />

Builds the app for future development to the `build/dev` folder.<br />
